# dotfiles
## Dependances
| Name  | Required  | Description  |
|:---|:---|:---|
| [Z shell](https://en.wikipedia.org/wiki/Z_shell)  | True  | The Z shell (Zsh) is a Unix shell that can be used as an interactive login shell and as a command interpreter for shell scripting.  |
| [GNU Stow](https://www.gnu.org/software/stow/)  | True  | GNU Stow is a symlink farm manager which takes distinct packages of software and/or data located in separate directories on the filesystem, and makes them appear to be installed in the same place.  |
| [Git](https://git-scm.com/)  | False  | Git is a distributed version control software.  |
| [FZF](https://github.com/junegunn/fzf#setting-up-shell-integration)  | False  | fzf is a general-purpose command-line fuzzy finder.  |
| [kitty](https://sw.kovidgoyal.net/kitty/)  | False  | kitty is a cross-platform terminal emulator that uses GPU and SIMD vector CPU instructions for best performance and graphics.  |
